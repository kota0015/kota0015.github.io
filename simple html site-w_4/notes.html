Some learned HTML Tags

The <html> tag:
Used as a basic tag to creat an HTML document.
The <cite> tag:
Used to a citation and usually processed in italic.
The <div> tag:
Used to format block content with CSS.
The <em> tag:
Used to emphasize a word ( usually processed in italic).
The <hr> tag:
Used to insert a horizontle rule.
The <address> tag:
Used to creat address section, usually processed in italic.
